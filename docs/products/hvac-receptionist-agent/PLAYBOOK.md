# Playbook — HVAC AI Receptionist

## Core rule

**Lead with the demo phone call, not the product.**

Email #1 must get them to call the regional demo line in the footer. The call sells better than copy.

## Sequence angle

1. **Email 1 — Demo first (missed calls)**
   - Subject: short, curiosity or local (e.g. "after-hours calls at [Company]?" or "quick test for [City] shops")
   - Open with one specific observation if known (reviews, after-hours, peak season)
   - Body: 3–4 sentences max. CTA = call the demo line and pretend AC broke Saturday night
   - Do NOT lead with "AI receptionist" or "$299/mo" in email 1
   - Footer has regional demo number — reference it explicitly in the body

2. **Email 2 — Peak season overload (only if no reply)**
   - Reference summer rush / dispatcher overload
   - Again: demo line CTA before pricing

3. **Email 3 — Breakup**
   - "Should I close the loop?"
   - One line: demo still available if curious

## Subject line patterns (A/B test these)

- `after-hours at [Company]?`
- `[City] HVAC shops — 30 sec test`
- `missed calls → booked jobs`
- `heard your line might go to voicemail`

## CTA hierarchy

1. Call demo line (primary)
2. 15-minute walkthrough (secondary)
3. Link to makola.org/hvac (tertiary)

## Objections

- "Customers want a human" → AI handles overflow and after-hours; humans handle complex jobs
- "Too expensive" → one booked emergency job often covers $299/mo
- "We have an answering service" → ask if it books jobs or just takes messages

## Do not

- Wall of text
- Generic "I help HVAC companies with AI"
- Multiple CTAs in email 1
- Promise features not in OFFER.md
